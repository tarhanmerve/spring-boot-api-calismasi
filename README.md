# CV Management System - Backend

Bu proje, Spring Boot ile geliştirilmiş bir API çalışmasıdır.

* **Teknoloji:** Java ve Spring Boot kullanılmıştır.
* **Mimari:** Kodlar Controller, Service ve Repository katmanlarına ayrılmıştır.
* **Önemli Not:** Bilgisayarımda oluşan teknik bir arıza nedeniyle projenin sadece mail üzerinden kurtarabildiğim kaynak kodlarını içermektedir.
